> **에이전트 사용 안내**
> 1. 이 템플릿을 `docs/<서비스명>/plans/HANDOFF.md`에 복사한다.
> 2. 세션이 종료될 때마다 최상단에 새 `## Session Update` 섹션을 추가한다.
> 3. 가장 최신 세션이 항상 문서 상단에 위치한다.
> 4. HANDOFF_NOW.md가 60줄을 초과할 경우 이 문서의 "이전 세션" 섹션으로 이관한다.
