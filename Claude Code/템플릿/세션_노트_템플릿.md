# 세션 노트 템플릿

`docs/<서비스명>/history/세션_노트.md` 에 위치하는 세션 요약 로그 초기화 템플릿.

> **에이전트 사용 안내**
> 1. 이 템플릿을 `docs/<서비스명>/history/세션_노트.md`에 복사한다.
> 2. 세션 종료 시마다 최상단에 한 줄 요약을 prepend한다.
> 3. **길게 쓰지 않는다** — 한 세션당 1~2줄. 상세는 HANDOFF.md에 있다.
> 4. 30일 이상 된 항목은 `history/HANDOFF_archive_YYYY.md`로 이관한다.
