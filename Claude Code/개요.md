# Claude Code 하네스 개요

개인 Claude Code 환경의 전체 구성을 정리한 문서.

## 디렉터리 구조

```
~/.claude/
├── CLAUDE.md              ← 전역 하네스 진입점 (모든 프로젝트에 로드)
├── settings.json          ← 전역 설정 (훅, 권한, 플러그인, additionalDirectories)
├── skills/                ← 스킬 패키지 모음
├── hooks/                 ← 훅 스크립트
├── agents/                ← 커스텀 에이전트 정의
└── projects/              ← 프로젝트별 컨텍스트 + 자동 메모리
    └── <프로젝트경로-슬래시를-하이픈으로>/
        └── memory/        ← 자동 메모리 (프로젝트별 독립)
            ├── MEMORY.md
            ├── feedback_*.md
            ├── project_*.md
            ├── reference_*.md
            └── user_*.md
```

> **메모리 경로 규칙**: `/Users/xxx/업무/myproject` → `-Users-xxx-업무-myproject`
> Claude Code가 프로젝트에서 첫 세션을 열면 `projects/` 하위 경로가 **자동 생성**된다.
> 수동 mkdir 불필요. MEMORY.md만 초기화하면 된다.

## 하네스 레이어 구조

Claude Code는 아래 순서로 CLAUDE.md를 누적 로드한다. 하위로 갈수록 범위가 좁아지며, 충돌 시 더 제한적인 규칙이 우선 적용된다.

```
~/.claude/CLAUDE.md                  ← 1. 전역 (모든 프로젝트 공통)
<project-root>/CLAUDE.md             ← 2. 프로젝트 (팀 공유, git 관리)
<project-root>/.claude/CLAUDE.md     ← 3. 로컬 (개인 설정, gitignore 권장)
```

## 핵심 파일

| 파일 | 역할 |
|------|------|
| `~/.claude/CLAUDE.md` | 전역 규칙 — 모든 프로젝트 세션에 자동 로드 |
| `~/.claude/settings.json` | 훅, 권한, 플러그인, Obsidian Vault 경로 등 설정 |
| `<project-root>/CLAUDE.md` | 프로젝트 전용 하네스 — 팀과 공유 |
| `<project-root>/.claude/settings.json` | 프로젝트 로컬 설정 — git 관리 여부 선택 |

## settings.json 주요 섹션

```json
{
  "permissions": {
    "allow": [],               // 자동 허용할 도구·경로
    "additionalDirectories": [] // Claude가 접근 가능한 추가 경로 (Obsidian Vault 등)
  },
  "hooks": {},                 // PreToolUse / PostToolUse 훅
  "enabledPlugins": {}         // MCP 플러그인 on/off
}
```

## 관련 문서

- `하네스/운영_가이드/코딩_원칙.md` — Claude Code 작업 4원칙 + STEP 0~3 운영 매핑
- `하네스/Claude Code/스킬 목록.md` — 설치된 스킬과 용도
- `하네스/Claude Code/훅 설정.md` — PreToolUse / PostToolUse 훅 동작
- `하네스/Claude Code/초기_설정_체크리스트.md` — 새 프로젝트 하네스 구축 순서 (Phase 0~10)
- `하네스/Claude Code/에이전트_정의_가이드.md` — 커스텀 에이전트 정의 방법
- `하네스/Claude Code/메모리_관리_가이드.md` — 자동 메모리 운영 방법
- `하네스/설정_템플릿/settings.json_템플릿.md` — settings.json 전체 구조 템플릿
- `하네스/설정_템플릿/mcp_설정_템플릿.md` — MCP 서버 등록 스니펫
- `하네스/Claude Code/트러블슈팅.md` — 설치·훅·/project-init 오류 해결
